Overview:

This use case allows the customer or the agent to obtain available frequent flier miles from the legacy miles system. The customer/agent selects view miles from the menu - the system responds with the available/redeemable miles. The sequence of this use case is captured in one sequence diagram.

Assumptions:

  • Direct access to the legacy award miles datastore is available.
  • User has a miles account.
